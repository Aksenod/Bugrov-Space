# ⚠️ ВАЖНО: Localhost больше НЕ используется для бэкенда

## Бэкенд работает ТОЛЬКО на Render

**Бэкенд API больше НЕ запускается локально на localhost:4000**

- Бэкенд работает только на **Render**: `https://bugrov-space.onrender.com`
- Фронтенд подключается к бэкенду через переменную `VITE_API_URL` в `.env`
- **НЕ пытайтесь запускать бэкенд локально** - это не поддерживается

## Конфигурация

### Фронтенд (.env в корне проекта)
```
VITE_API_URL=https://bugrov-space.onrender.com/api
```

### Бэкенд (на Render)
- `NODE_ENV=production` - обязательно!
- `CORS_ORIGIN` - не обязателен, если `NODE_ENV=production` (разрешает все origins)

## Если нужно изменить URL бэкенда

1. Обновите `VITE_API_URL` в `.env` файле
2. Перезапустите dev-сервер фронтенда
3. **НЕ создавайте локальный бэкенд** - используйте только Render

## История

Ранее бэкенд мог работать локально на `localhost:4000`, но это больше не поддерживается.
Все запросы идут напрямую на Render.
